## Setup

Setup [SSH](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) before clone:

```bash
git clone git@github.com:lurc-zmei/php-project-45.git
cd php-project-45

make install
```

### Run linter or fix

```sh
make lint

make lint-fix
```
